# Code of Conduct

StackSpend aims to be a practical and respectful open-source project.

## Expected behavior

- Be respectful and constructive.
- Assume good intent.
- Keep discussions focused on the project.
- Avoid sharing secrets, credentials, or personal data.
- Use synthetic examples when reporting bugs.

## Unacceptable behavior

- Harassment or personal attacks
- Publishing another person's private data
- Posting real API keys, tokens, account identifiers, or billing data
- Encouraging unsafe provider write actions outside the project scope

## Enforcement

Maintainers may edit, hide, or remove comments that expose secrets or personal data. Maintainers may close issues that cannot be safely discussed publicly.
